## 🚀 DEPLOY TO VERCEL

### Method 1: Drag & Drop (EASIEST)

1. Go to [vercel.com](https://vercel.com)
2. Sign up with email/GitHub
3. Click "Add New..." → "Project"
4. **Drag the entire `medvoyage-final` folder** onto the screen
5. Click "Deploy"
6. Done! Your site is live!

### Method 2: GitHub + Vercel (AUTO-DEPLOY)

**Step 1: Upload to GitHub**

1. Go to [github.com](https://github.com)
2. Create new repository: `medvoyage-website`
3. Upload all files from `medvoyage-final` folder
4. Commit files

**Step 2: Connect to Vercel**

1. Go to [vercel.com](https://vercel.com)
2. Click "Add New..." → "Project"
3. Click "Import Git Repository"
4. Select your `medvoyage-website` repository
5. Framework: **Other**
6. Root Directory: `./`
7. Click "Deploy"
8. Done!

**Future Updates:**
- Just update files in GitHub
- Vercel auto-deploys instantly!
